
## 🚀 How to Use

1. **Download** the `.pbix` file from this repository
2. **Open** with Power BI Desktop (free version available from Microsoft)
3. **Connect** to your data source or refresh with new data
4. **Explore** the interactive visualizations and filters

## 📌 Requirements

- Power BI Desktop (October 2023 or later recommended)
- Windows 10/11 or macOS (via Parallels/Bootcamp)
- Basic understanding of Power BI navigation
